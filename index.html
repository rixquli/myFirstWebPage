HTML

Pour le bien de l'explication les commandes seront faussés c'est a dire que des points seront placés juste après les <, donc faites attention si vous copier coller.
Si vous avez tous compris, alors dans ces cas là allons-y

Pour commencer toute page web il faut ecrire ceci:
<.DOCTYPE html>
<.html>
<.head>
<.meta charset="utf-8"/>
<.title>Titre<./title>
<./head>
Remplassez Titre par le titre de votre page
Puis, il faut que vous ouvriez la balise body:
<.body>
C'est a l'interieur que vous ecrirez la page, le texte.
Pour ecrire une page vous devez utilisez des balises qui se trouve entre < et >.

liste balise utilisable pour commancer:

  • <.p> et <./p>
  • <./br>
  • <.a>
  • <.dl>
  • <.ol>
  • <.ul>

<.p> et <./p>

Par exemple vous pouvez utilisez <.p> et pour la fermer <./p>. Ces commandes permettent d'ecrire dans un paragraphe. Entre ces deux symboles ecrivez du texte. Par exemple:
<.p>Ceci est un paragraphe<./p>
<.p>Ceci est un autre paragraphe<./p>

<./br>

En ecrivant vous ne pouvez pas faire de retours a la ligne, donc <./br> est la solution. Il s'utilise partout et où vous voulez.

<.A>

La balise a permet d´introduire un autre site où d´ammener sur un endroit de la page. Il s´utilise comme ceci:
<.a href="url.com">
ou alors pour amener sur un endroit de la page il faut qu´a un endroit autour d'un mot vous utilisez les balise h1-h2-h3...h6 ou la balise <.div>. Vous devrez alors faire ceci:
<.h3 id="test">on vous fera voyager ici<./h3>
<.a href="#test">ceci vous fera voyager><./a>

<.dl>

DL permet de creer une sorte de liste mais sans numerotation, ni point a chaque retour a la linge. Il s´utilise ainsi:
<.dl>
<.dt>Titre de la liste<./dt>
<.dd>element de la liste<./dd>
<.dd>et un deuxieme<./dd>
<./dl>

CSS

Le CSS permet de modifier l'apparence d'une page par exemple la couleur du texte.

Le CSS se presente sous cette forme

Ce qu´on veut changer par exemple tous la couleur d´un paragraphe alors on utilise:
p{
color:red;
}

le P correspont a ce qu'on veut modifier, ausi dans une balise on peut ecrire une "class" au debut par exemple <./p class="classe1"> par la suite en CSS, vous devrez remplacer le P par .classe1 par exemple. Il existe d´autre moyenne pour modifier le texte comme:

  • front
  • color
  • opacity
  • text
  • background
  • position
  • top
  • bottom
  • left
  • right

Ce sont les principales variables utilisable en CSS, pour modifier le texte.
